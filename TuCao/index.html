# 我收集的一些片段

我儿子说《铁棒磨成针》是毒鸡汤，认为老奶奶应该把铁棒卖了去买针，该怎么教育？

李白小时候不好好读书，出去玩的时候遇到一位老奶奶要把铁杵磨成针。李白看了之后大受震撼，从此就开始认真学习，因为：他知道如果没有知识，就会像这位老奶奶一样蠢。

《三体》里最被低估的人是谁？

“各位，我想到件事，你看，如果我们想办法向外面传递信息……”不不不，任何这样的交流，都会被智子监听。只要这想法一说出口，就意味着它的破产。那要怎么办呢？“我们来做些什么，所有人，听我说，让我来办，我有办法。”不不不，这样说的话，其一，你可能没有这样的威望，毕竟，别人都想着办IMPART了，你还说有什么办法——却无法将这个办法描述出来，那么，别人未必听你的。其二，你不确定同僚们是否也想到了和你相同的事，假如他们没想到呢？凭你一个威望不足的小官员，又如何调动资源做这件事？所以，你思前想后，说出了这样的话：“各位同事，事已至此，我们总得做些什么。我建议，咱们可以在太阳系这样广阔的空间里，做些文章。”“能做什么呢？现在做什么都没用了。”这时，有同事答复你说：“或许我们可以做一些核弹，当它爆炸时，产生星际尘埃……”你看着他，发现他以同样灼热的目光望向你。你们不能进行任何交流，但，彼此的目光里，似乎有了点默契。你不确定，这样的默契到底是不是你的错觉，或许，他只是随口说两句而已。“我们这样做有什么意义呢？就算产生了尘埃，也只能定位三体舰队而已。”另一位同事站出来说。他的表情满是沮丧，但他的眼神里，似乎有光。“如果有足够多的核弹呢？”你小心翼翼地试探。“不过是更大的尘埃云而已。”那位同事说着泄气的话，却似乎在向你点头。“总好过什么也不做吧。我们……”你思考着，该如何说，才能让别人相信，或者说，让三体人相信：“我们……总需要一些事，来安抚民众。”“只不过是无谓的挣扎。”有人离开了会议，再也没有回来。也有人抬起头，望向了你。“我们总得维持人类最后的尊严。”你望向众人，发现其中有些人，也在坚定地回望你。你不知道他们在想什么，这些人的思维对你来说是封闭的，不透明的，正如对三体人那样。“那就这样办吧，让谁来领导这项工程呢？”你问。这是个难题。你不能简单地将工程指挥权托付给这些同事中的任何一位，因为你虽然感觉到了，他们可能想得和你一样——但是，你无法确定。万一呢，我是说，万一呢！同样的道理，你也无法确定，他们是否觉得，你想的，和他们的一样。在他们的角度看来，困境是相同的。万一呢？所以，他们也不会托付给你。谁能行？你和同事们陷入了沉默，但你想到了还有一个人，是可靠的。罗辑。作为咒语的发起人，罗辑，他必然是懂的。虽然他宣称那是条咒语，但拜托，谁都不是小孩子。咒语生效了，证明了一切。你可能不知道它生效的原理是什么，但你已经很确定了，只要能够发出恒星的坐标信息，那颗恒星就会被摧毁。这就是你们的最后希望，很显然，罗辑是最清楚这一点的人了。更妙的是，只要是和你想得一样的人，他们也都会认同，罗辑是懂的。你们不需要交流，已有的事实，就足以让你们达成一致。“谁来领导雪地工程？”有人问。他望着你，目光里充满鼓励。他想得和你一样吗？你不确定。你只能抓住这次机会，说出你的人选：“罗辑博士。”你打算说更多话，来完善你的推荐。比如，他是曾经的面壁者，比如，他的咒语生效了。你还没有说出来，同事们就点了头：“好，我们也是这样想的。他有足够大的威望。”你望着他们，正如，他们也正望着你。在这一刻，新的面壁者（们），诞生了。“工程细节呢？”终于到了最实质的内容，毕竟，那些核弹要如何摆放，如何引爆，这才是关键。“全权交给罗辑博士。”有同事提议。他顿了顿，补充说：“我们或许该恢复他的面壁者身份。”“全权交给他？可是他并没有相关知识背景，无法高效地布置核弹的位置，造成资源的浪费。”这位同事看起来是在反对，但好像，又在为罗辑留出足够的操作空间。你不确定他的真实想法，只能顺着他的话说下去：“我们可以调一些相关领域的专家来协助罗辑博士的工作。但别忘了，雪地工程本质上只是为了安抚民众，维持人类最后的体面。你我都不期待它能真的对未来抵抗三体舰队产生什么作用。所以，核弹要怎么放，就随他去吧。”那个人耸了耸肩，没有表示同意，但也没有反对。这事就这样定下来了。
