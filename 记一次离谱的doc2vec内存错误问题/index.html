记一次离谱的doc2vec内存错误问题

发表于 2023-10-17
  • Aki's Memory
首页 代码杂谈

# 起因

我在编写一个类似 chatterbot 的工具，当时要标识每一句话的，第一反应就是用雪花生成一个 id，然后去 doc2vec 中训练时使用雪花 id 作为 tag

然后就报错了，显示的是 Memmory Error，错误描述没有，只有错误位置。

错误的位置就是这里

# 瞎猜原因

其实如果我不用 jupyter 看见上面 logger 的 warning 的话可以很快才出来原因是什么的，可惜我用了 jupyter 没有看到 log 给的 warning。

dec2vec 把输入的 TaggedDocument 类型中的 int 或者 integer 类型的数据当成了数组的索引，所以直接报错了，转成 string 类型就好。

# 解决方式

解决方法很简单，不用雪花就好或者转成 string 形式就好，雪花生成的九位数 id 对于 dec2vec 来说太大了，推测是他看数据集的大小看的是标签的大小，一看过亿了，直接报错。
