# Windows 系统美化 [用于 win10 & 11]

# 首先是我的桌面效果

# 我用到的软件

  • wallpaper engine
  • rainmeter
  • bitdock
  • start11
  • files
  • coodesker

# 开始

# 鼠标指针

鼠标指针 - 光标 - 电脑鼠标指针下载 - 致美化 - 漫锋网 (zhutix.com)

  1. 选择自己喜欢的指针包下载
  2. 解压
  3. 来到存放指针图标的文件夹
  4. 回看见一个以 inf 结尾的文件
  5. 右键选择安装，如果右击没用安装那就是在显示更多里面
  6. 确定
  7. 安装完成就会看见你鼠标样变了

# 安装软件

# files

安装 files, 并且赋予管理员权限

虽然 files 比文件资源管理器好看，但是千万别设置成默认的，恢复很麻烦

微软商店可以直接下载

# start11

Start11 是为 Windows 11 和 Windows 10 系统设计的，并装载了旨在使 Windows 更加个性化和高效的功能。 它支持将 “ 开始 “ 菜单按钮移到中间或左边，调整任务栏大小，重新排列快速访问快捷方式列表，网格间距选项，增强经典和现代搜索体验，恢复任务栏上下文菜单等。

Start11 下载地址 - Stardock-Start11

# wallpaper engine 动态背景

steam 商店或者官网下载

不过 wallpaper engine 是收费软件，但是只要 18 块钱

建议使用 steam 版

创意工坊中有数不尽的壁纸，绝大部分你想要的都能找到，如果没有，跟我说，我做

而且完全可以把创意工坊当一个免费云盘，还不限量不限速，所里里面有很多好东西

如果不想掏钱可以用 upupoo, 不过壁纸资源少一点

如果下载 steam 注意别被百度坑了

欢迎来到 Steam (steampowered.com)

# Cooderker 桌面整理

桌面文件分离整理，支持文件夹映射等，是个很方便的桌面整理软件

酷呆桌面 - 桌面整理工具 (coodesker.com)

# BitDock dock 栏

原本我是没有用 bitdock 的，但是前段时间升级成 win11 之后发现 win11 任务栏不能放侧边，所以选择了 bitdock

相似功能的软件有很多，包括 rainmerter 也可以实现，但是处于美观考虑我选择了 bitdock

任务栏可以设置成自动隐藏，然后主要使用 dock 栏，个人推荐放在侧边

BitDock 比特工具栏 - 官方网站

# 最后是 Rainmeter

rainmeter

Rainmeter 是 Windows 平台上最知名最流行的桌面个性化定制工具。

Rainmeter 皮肤能够在工作和学习中提高你的效率，这些皮肤浮动在你的桌面上，你可以很方便的看到 Rainmeter 皮肤为您提供的信息。

系统资源，内存，电池（电源），网络数据流量，电子邮件，RSS 源，天气等，这些信息一目了然。

rainmeter 只是一个桌面插件，他有很多皮肤，但是需要自己动手去选择合适的皮肤配合合适的背景才能达到完美的效果，所以我放在了最后，其实 ye 也可以尝试自己写一些桌面插件，我用的其中两个插件就是自己写的

[RainMeter 使用教程一篇_iteye_2225 的博客 - CSDN 博客](https://blog.csdn.net/iteye_2225/article/details/82304110#:~:text=Rainmeter 是 Windows 平台上最知名最流行的桌面个性化定制工具。 Rainmeter 皮肤能够在工作和学习中提高你的效率，这些皮肤浮动在你的桌面上，你可以很方便的看到 Rainmeter 皮肤为您提供的信息。,RainMeter （截止 2012%2F4%2F1 ， RainMeter 的最新稳定版本是 2.2 ）)
